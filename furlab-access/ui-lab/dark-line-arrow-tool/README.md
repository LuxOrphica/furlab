# dark-line-arrow-tool

Мини-проект: загрузка изображения, поиск тёмной (чёрной) линии со стрелкой и построение контура светлой зоны по кнопке.

## Запуск

Откройте `index.html` в браузере.

Для локального сервера из корня репозитория можно запустить:

```powershell
npx serve ui-lab/dark-line-arrow-tool
```

## Как работает

1. Изображение рисуется в `canvas`.
2. Считается яркость каждого пикселя.
3. Берётся заданный процент самых тёмных пикселей как маска линии.
4. Из тёмной маски выделяются связные компоненты.
5. Выбирается наиболее «линейная» компонента (по вытянутости и форме).
6. Через PCA оценивается направление, и по главной оси рисуется стрелка.
