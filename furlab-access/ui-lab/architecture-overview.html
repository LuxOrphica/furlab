Архитектура FURLAB (React + Legacy Engine + Access API)

Текущая рабочая схема: React-UI управляет состоянием, legacy-ядро делает вычисления, Node API пишет в Access.

1) UI слой (React + Ant Design)

App.tsx, uiState, useLegacyBridge. Рисует интерфейс, отправляет команды движку, принимает состояние через события.

2) Вычислительный слой (Legacy appCore.js)

Canvas, QR, маски, P1-P2, расчет метрик и валидации. Публикует ldv:state и ldv:save-status.

3) Сервер API (Node)

tools/ui_lab_server.js: /api/health, /api/dicts, /api/save-scrap-piece.
